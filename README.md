# Bank-System
This project showcases many skills that employers look for in a Java backend developer: REST API design, layered architecture, Spring Boot, Spring Security with JWT, JPA/Hibernate, MySQL, exception handling, unit testing with JUnit 5 and Mockito, and clean code organization. 
# Bank Management System

A production-ready Banking System backend developed using Java and Spring Boot following layered architecture and REST API best practices.

## Features

- User Management
- Account Management
- Deposit Money
- Withdraw Money
- Fund Transfer
- Transaction History
- Balance Inquiry
- JWT Authentication
- Role-Based Authorization
- Global Exception Handling
- Request Validation
- Logging
- Unit Testing using JUnit 5 & Mockito

## Tech Stack

- Java 21

- Spring Data JPA

- PostgreSQL
